# Замена байтовых последовательностей в файле

Программа для поиска и замены произвольных байтовых последовательностей в файле.
Чтение осуществляется блоками заданного размера.

## Сборка

Для сборки проекта требуется CMake и компилятор C (например, GCC/MinGW).

```bash
mkdir build
cd build
cmake -G "MinGW Makefiles" -DBLOCK_SIZE=1024 ..
cmake --build .
